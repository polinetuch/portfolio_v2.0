The Liri Node

  • View Code
  • Click here to see a demo video
    When starting the node server on node liri.js, the app will begin listening to the following commands:
  • concert-this
  • spotify-this-song
  • movie-this
  • do-what-it-says
    Here are a few command line examples:
  • node liri.js concert-this U2
  • node liri.js spotify-this-song Beautiful People
  • node liri.js movie-this Ponyo
    Technology used:
  • Javascript library: Node Js
  • Bands In Town API
  • Spotify API
  • OMDB API
